# SportIQ — Образовательная платформа с тестами по спорту

Выпускной проект по дисциплине «Веб-дизайн»

## Описание

Интерактивная платформа для прохождения тестов по спорту с таймером, подсчётом баллов и отображением прогресса. 4 уровня сложности, 80 вопросов.

## Функционал

- ⏱ **Таймер** — обратный отсчёт на каждом тесте (10/8/7/6 минут)
- 📊 **Подсчёт баллов** — автоматический подсчёт правильных ответов
- 📈 **Прогресс** — полоса прогресса по вопросам
- 💾 **Сохранение** — результаты в localStorage
- 📱 **Адаптивный дизайн** — корректно на всех устройствах
- 🎨 **Спортивный стиль** — тёмная тема, неоновые акценты

## Тесты

| Уровень | Название | Вопросы | Время |
|---------|----------|---------|-------|
| 🌱 Начинающий | Основы спорта | 20 | 10 мин |
| ⚡ Средний | Спортивные правила | 20 | 8 мин |
| 🔥 Продвинутый | История и рекорды | 20 | 7 мин |
| 👑 Эксперт | Олимпиада и мировой спорт | 20 | 6 мин |
